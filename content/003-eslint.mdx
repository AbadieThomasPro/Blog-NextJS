Bonjour !

Dans cet article, découvrons pourquoi **ESLint** est devenu un outil incontournable pour les développeurs travaillant avec JavaScript ou TypeScript.

## Qu'est-ce qu'ESLint ?

ESLint est un **linter** JavaScript. Un linter est un outil qui analyse votre code pour identifier des erreurs potentielles ou des pratiques de codage non conformes à vos standards.

### Pourquoi utiliser ESLint ?

- **Détection d'erreurs :** Identifie les bugs courants avant même l'exécution du code.
- **Conventions :** Uniformise le style de code au sein de l'équipe.
- **Support TypeScript :** Parfait pour les projets modernes avec TypeScript.
- **Extensibilité :** Large choix de plugins pour adapter ESLint à vos besoins (React, TailwindCSS, etc.).

## Et comment l'intégrer chez toi ?

### Étape 1 : Installer ESLint

Lance cette commande dans ton projet :
```bash
npm install eslint --save-dev
```

### Étape 2 : Initialiser la configuration

Crée un fichier de configuration **.eslintrc** en lançant :
```bash
npx eslint --init
```
