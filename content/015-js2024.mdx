## Introduction

JavaScript, le langage incontournable du web, ne cesse de s'améliorer avec chaque nouvelle version. En 2024, la dernière mise à jour apporte des fonctionnalités attendues et des optimisations qui simplifient la vie des développeurs.

## Les nouveautés majeures de JavaScript en 2024

### **1. Pattern Matching**

Inspiré de langages comme Python ou Haskell, **le pattern matching** fait enfin son apparition.  
Il permet une manipulation plus intuitive et expressive des structures complexes, comme les objets ou les tableaux.

```javascript
const fruit = { type: "apple", color: "red" };

switch (fruit) {
  case { type: "apple", color: "red" }:
    console.log("C'est une pomme rouge !");
    break;
  default:
    console.log("Fruit inconnu");
}
```
