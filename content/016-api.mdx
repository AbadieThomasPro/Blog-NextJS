Les API Web jouent un rôle fondamental dans le développement moderne, permettant aux applications de communiquer entre elles et d'exploiter les fonctionnalités des navigateurs de manière puissante et efficace.

## Qu'est-ce qu'une API Web ?

Une API Web (Application Programming Interface) est un ensemble de fonctionnalités ou d'outils qui permettent aux développeurs d'interagir avec des services ou des fonctionnalités spécifiques via des requêtes standardisées. Elles permettent d'ajouter de la complexité et des capacités à une application sans avoir à développer tout de zéro.

Exemple : Utiliser l’API Fetch pour récupérer des données d’un serveur sans avoir à écrire un client HTTP complet.

## Pourquoi sont-elles importantes ?

1. **Interopérabilité** : Les API Web permettent aux systèmes différents de fonctionner ensemble en fournissant un langage commun.
2. **Réduction des coûts et du temps de développement** : En utilisant des API préexistantes, les développeurs peuvent se concentrer sur la logique métier.
3. **Accès simplifié à des fonctionnalités puissantes** : Par exemple, accéder au micro ou à la caméra via l’API WebRTC.

## Les API Web principales

Voici un aperçu de quelques API Web importantes :

### 1. **API Fetch**
Permet d'effectuer des requêtes réseau, comme récupérer des données depuis une API tierce ou envoyer des données à un serveur. Elle remplace largement l'ancienne API `XMLHttpRequest`.

```javascript
fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => console.log(data));
```
### 2. **API DOM**
Offre un accès et une manipulation complète du Document Object Model (DOM) d'une page Web. Parfait pour manipuler les éléments HTML et CSS en temps réel.

```javascript
document.querySelector('#button').addEventListener('click', () => {
  alert('Bouton cliqué!');
});
```

### 3. **API Web Storage**
Permet de stocker des données localement dans le navigateur avec des options comme localStorage et sessionStorage.

```javascript 
localStorage.setItem('username', 'JohnDoe');
console.log(localStorage.getItem('username'));
```

### 4. **API WebRTC**
Facilite les communications en temps réel via les navigateurs, comme les appels vidéo et audio ou le partage de fichiers.

```javascript 
const peerConnection = new RTCPeerConnection();
```

### 5. **API Geolocation**
Utilisée pour obtenir la localisation géographique de l’utilisateur (avec son consentement).

```javascript 
navigator.geolocation.getCurrentPosition(position => {
  console.log(`Latitude: ${position.coords.latitude}`);
  console.log(`Longitude: ${position.coords.longitude}`);
});
```

### 6. **API Canvas**
Offre la possibilité de dessiner graphiquement dans un <canvas/> HTML, utile pour les jeux et animations.

```javascript 
const canvas = document.getElementById('myCanvas');
const ctx = canvas.getContext('2d');
ctx.fillStyle = 'blue';
ctx.fillRect(10, 10, 100, 100);
```

### 7. **API Notifications**
Affiche des notifications natives aux utilisateurs via le navigateur.

```javascript 
if (Notification.permission === "granted") {
  new Notification("Hello! Voici une notification.");
}
```
